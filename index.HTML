1. Culture Manual





글씨가 변화하면서 페이지 로딩이 시작되고, 로딩이 끝나면 회사의 소개, 가치관등이 쭉 설명되어있다.

재밌었던 점은 한 단어에 마우스를 가져다 대면 동영상이 재생되는데 페이지에 이미지가 많이 없었지만 지루하지않았다.


2. Zero Studio





글줄로 시작하는 첫화면에서 내리다보면 이 스튜디오의 작업 이미지를 볼수있는데, 이때 효율적으로 옆으로 넘기며 볼 수 있도록 해놓았다.

타이포의 위계도 보기 좋고 페이지 마지막에 넣은 재밌는 이미지도 전체적으로 균형있다고 생각하여 선정하였다.



3. Melville





이미지와 타이포그래피가 조화롭게 잘 이루어져있고

브랜드의 아이덴티티와 분위기를 쉽게 이해할 수있게 만들어 놓은것 같아 선정하게되었다.



4. Rocca fiore wines





스크롤을 내리면서 아이콘을 클릭하면 바로바로 볼 수있는 제품정보나

기업의 소개가 인상적이었고 편리하다고 생각되어 선정했다.


내가 만들고 싶은 홈페이지

내가 만들고 싶은 홈페이지는 3. Melville와 같은 페이지이다. 다른 페이지보다 타이포와 레이아웃이 더 중점이 되어보인다.

미세하게 조정할 부분이 많아 코딩하기 조금 까다로워 보이지만 그런 구현 연습이 나에게 많은 도움이 될 것 같다.
